# hazeteam-openclaw

OpenClaw integration packages for Hazeteam.

Current primary package:

- `packages/openclaw-adapter`

Current scope:

- contracts and repository foundation only;
- no production Telegram bot/runtime wiring yet;
- no real OpenClaw runtime API wiring yet.

This repository was promoted from `hazeteam-core` staging planning into a dedicated `hazeteam-openclaw` workspace.

## Commands

```bash
npm ci
npm run build
npm run test
npm run check
```
