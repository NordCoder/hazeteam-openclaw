# @hazeteam/openclaw-adapter

TypeScript package for OpenClaw/Telegram adapter code over public `hazeteam-core` APIs.

Current status: package metadata plus shared adapter-owned contracts for refs, operation context, and safe result/error primitives.

This package does not contain OpenClaw Telegram event mapping, topic binding, rendering, delivery, callbacks, runtime bridge, approval bridge, durable stores, or real OpenClaw/Telegram SDK wiring yet.
